# Prompt 1 - Child Theme WooCommerce tienda ropa deportiva

**Stack:** PHP, WordPress, WooCommerce, CSS, JavaScript

---

Crea un child theme de WooCommerce para tienda de ropa deportiva chilena con integraciones reales. Todo lo de la version A mas: 1) integracion con plugin Transbank WebPay para WooCommerce (gratuito, configurar con credenciales de integracion), 2) integracion con Chilexpress API para calculo de costos de envio en tiempo real segun comuna de destino, 3) validacion de RUT chileno con algoritmo de digito verificador, 4) integracion con WooCommerce REST API para que un panel externo consulte pedidos. Incluir instrucciones de instalacion en WordPress local (Local by Flywheel o Docker), instalacion del plugin Transbank y configuracion de credenciales sandbox.
